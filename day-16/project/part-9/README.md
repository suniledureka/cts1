
## Register a user

```
POSTMAN --request POST 'http://localhost:8080/auth/register' 

--data-raw '{
    "name":"sunil",
    "password":"12345",
    "email":"sunil@gmail.com"
}'

```

## Generate token

```
POSTMAN--request POST 'http://localhost:9898/auth/token'

--data-raw '{
    "username":"sunil",
    "password":"12345"
}'
```
